# 示例：用 ASTRAION 构建一个 CRM 系统

> 展示管理员如何通过对话快速搭建一个完整的客户关系管理系统。

---

## 场景
作为管理员，我需要一个 CRM 系统来管理客户、联系人、商机和合同。

---

## Step 1: 创建客户模型

```
管理员：帮我建一个客户管理

AI：好的，我来创建"客户"模型。一个典型的客户包含以下信息：
     ✅ 客户名称（必填）
     ✅ 联系电话
     ✅ 邮箱
     ✅ 客户等级：普通 / 重要 / VIP
     ✅ 所属行业
     ✅ 地区
     ✅ 状态：跟进中 / 已成交 / 已流失
     
     还需要其他字段吗？

管理员：再加个"来源渠道"和"备注"

AI：好的，已添加。最终配置确认：

┌───────────── 模型：客户 ──────────────┐
│ 1️⃣ 客户名称（文本，必填，唯一）       │
│ 2️⃣ 联系电话（手机号）                 │
│ 3️⃣ 邮箱（邮箱格式）                   │
│ 4️⃣ 客户等级（枚举：普通/重要/VIP）    │
│ 5️⃣ 所属行业（文本）                   │
│ 6️⃣ 地区（文本）                       │
│ 7️⃣ 来源渠道（枚举：线上/转介绍/展会）  │
│ 8️⃣ 备注（长文本）                     │
│ 9️⃣ 状态（枚举：跟进中/已成交/已流失）  │
│                                       │
│ [确认创建] [修改]                      │
└───────────────────────────────────────┘

管理员：确认

AI：✅ 客户模型已创建！
    - 数据库表自动建好
    - 立即可以通过对话管理客户了
    - 试试说：
      「添加一个新客户：张三，VIP，来自转介绍」
      「显示所有VIP客户」
```

---

## Step 2: 创建联系人模型

```
管理员：帮我建一个联系人管理，关联到客户

AI：好的，我来创建"联系人"模型。会自动关联到客户：

┌───────────── 模型：联系人 ────────────┐
│ 1️⃣ 姓名（文本，必填）                 │
│ 2️⃣ 手机号（手机号）                   │
│ 3️⃣ 邮箱（邮箱格式）                   │
│ 4️⃣ 职位（文本）                       │
│ 5️⃣ 所属客户（关联 → 客户模型）        │
│ 6️⃣ 是否为决策人（布尔）               │
│ 7️⃣ 备注（文本）                       │
│                                       │
│ [确认创建] [修改]                      │
└───────────────────────────────────────┘

管理员：确认

AI：✅ 联系人模型已创建！
    - 已自动关联到客户模型
    - 查看客户详情时可以看到关联的联系人
    - 试试说：
      「给客户张三添加一个联系人：李经理，手机138...」
      「显示张三的所有联系人」
```

---

## Step 3: 配置权限

```
管理员：客户和联系人只有销售部的人能查看和编辑

AI：好的，我来配置权限规则：

┌────────── 权限配置 ──────────┐
│ 模型：客户                    │
│ ✅ 查看 → 角色：sales         │
│ ✅ 创建 → 角色：sales         │
│ ✅ 编辑 → 角色：sales         │
│ ✅ 删除 → 角色：admin         │
│                              │
│ 模型：联系人                  │
│ ✅ 查看 → 角色：sales         │
│ ✅ 创建 → 角色：sales         │
│ ✅ 编辑 → 角色：sales         │
│ ✅ 删除 → 角色：admin         │
│                              │
│ [确认] [修改]                 │
└──────────────────────────────┘

管理员：确认

AI：✅ 权限配置已生效！
    - 非销售部人员无法查看客户和联系人数据
```

## 总结

```
传统方式搭建 CRM：
  设计数据库 → 写 Entity → 写 Repository → 写 Service →
  写 Controller → 写前端页面 → 部署 → 完成
  耗时：2周以上

ASTRAION 搭建 CRM：
  说话 → 确认 → 完成
  耗时：3分钟
```
